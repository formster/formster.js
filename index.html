formster.js - the Formster.io Client-side Library

Formster.io is a service to let you easily create and track sign up forms on static web pages. To sign up, visit Formster.io. For more information on the client-side library including installation and usage instructions, view the project on GitHub.

Example Form

To create a Formster.io form, you need to create a new project on Formster.io, like this public project. After a project has been created, you just need to add the <form> and <script> tags provided.

As an example, I might use the code below to produce a sign up form:

<style type="text/css">
  .thanks {
      display: none;
  }

  .formster.formster-submitted {
      display: none;
  }

  .formster.formster-submitted + thanks {
      display: block;
  }
</style>

<h3>Example Sign Up Form</h3>

<form class="formster" action="http://formster.io/submit?key=534314954dc4060b00000001" method="POST">

  <label>First Initial</label><br>
  <input type="text" name="f_initial"><br><br>

  <label>Last Name</label><br>
  <input type="text" name="last_name"><br><br>

  <label>Zip Code</label><br>
  <input type="text" name="zip"><br><br>

  <input type="submit" value="Sign Up">

</form>

<div class="thanks">
  Thanks for trying out the demo! Now check out the <a href="http://formster.io/projects/public/534314954dc4060b00000001">results</a>.
</div>


<script src="//cdn.formster.io/v0.4.1/formster.min.js"></script>

Which would produce the form below. Go ahead and try it out, and don't forget to check out the results page afterward.
